                              Welcome to Awk-estra
                              
      
                              
        A hands-on playground for running awk on text — paste in a
        log, CSV, or code file, then transform it with saved snippets, ad-hoc
        programs from the command palette, or multi-step pipelines you can
        save as reusable chains.
      
                              

      
                              Three ways to transform text
                              
      
                              
        No selection? The whole buffer is used as input.
      
                              
      
                                
        
                              1. 
          Quick transforms (Ctrl+K)
          — Select some lines in the editor, press the shortcut, type a short
          awk program, and hit Enter to replace the selection. Example:
          { print toupper($0) }.
        
                                2. 
        
                              2. 
          Saved snippets
          — Click a snippet in the sidebar to run it on the current selection.
          Ctrl+click runs it with no input
          and inserts the output at the cursor — handy for snippets that
          generate text (e.g. a BEGIN-only program).
          Hover a row for Edit / Duplicate / Add-to-pipeline / Delete. Click
          the ★ to favourite for quick access.
        
                                3. 
        
                              3. 
          Pipelines & chains
          — Click → on snippets to stack them
          into the pipeline panel below the editor. Run the whole pipeline
          with Run & Apply, then save it as a reusable chain.
        
                                4. 
      
                              

      
                              Keyboard shortcuts
                              
      
                              
        
          
                              Ctrl+KOpen the command palette (ad-hoc awk). Press again while open to toggle advanced / simple view.
                              
          
                              Ctrl+OOpen the Snippet Runner — type a snippet name, Enter to run on selection.
                              
          
                              Ctrl+Shift+OOpen the Snippet Runner in insert-at-cursor mode — output is inserted at the cursor (no input is fed in).
                              
          
                              Ctrl+FFind in editor
                              
          
                              Ctrl+HFind & replace
                              
          
                              Ctrl+GGo to line
                              
          
                              Ctrl+ZUndo (native textarea stack)
                              
          
                              Tab / Shift+TabIndent / outdent (multi-line selection: block; otherwise insert/remove a tab)
                              
          
                              Esc then TabMove focus out of the editor (one-shot — captures Tab again on next focus)
                              
          
                              EscClose the open palette / find panel
